#  Funcionalidades del sitio

Web diseñada para que los invitados puedan interactuar y acceder a toda la información fácilmente:

### 📍 Ubicación y Cómo Llegar
- Visualiza la dirección exacta del lugar.  
- Encuentra instrucciones claras para llegar sin complicaciones.  
- Mapa interactivo disponible para mayor comodidad.

### 🍽️ Elección de Platillo
- Selecciona tu platillo favorito entre varias opciones .  
- El sistema permite elegir según tus preferencias antes de la cena.  

### 🍷 Selección de Bebidas
- Escoge la bebida que acompañará tu cena.  
- Planificación anticipada para que todos disfruten su velada al máximo.  

### 📸 Galería de Fotos y Videos
- Explora recuerdos de su historia juntos.  
- Galería organizada en fotos y videos.

### ✅ Confirmación de Asistencia
- Sistema sencillo para confirmar asistencia online.
